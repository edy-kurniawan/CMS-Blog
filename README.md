## Requirement
1. Node.js
2. NPM
3. MySQL Server

## Installation
1. Run ```npm install```
2. Set the configuration DB name database_development.
3. Run the migration ```npx sequelize-cli db:migrate```.

## Running
To run this app, use ```npm run dev```

## EXSAMPLE 
```http://localhost:3000/api/users```

## FRONTEND
```http://localhost:3000```
